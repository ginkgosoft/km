# Filter

**1) Mail Status:** Filter status send Email to Buyer, click  ![](<../../.gitbook/assets/image (233).png>) as the picture below,

<figure><img src="../../.gitbook/assets/image (137).png" alt=""><figcaption><p>Filter – Mail status</p></figcaption></figure>

**Table** ‎: Mail status

| **Status**   | **Explanation**                                   |
| ------------ | ------------------------------------------------- |
| Not Send     | Items that haven't been emailed to the buyer yet. |
| Send Success | Items successfully emailed to buyers.             |
| Send Failure | Items that failed to send emails to buyers.       |
| Don’t Send   | Items that do not need to be sent to the IRS.     |



2\) **SMS** Status Filter status send SMS to Buyer, click ![](<../../.gitbook/assets/image (172).png>) as  the picture below,

<figure><img src="../../.gitbook/assets/image (167).png" alt=""><figcaption><p>Filter – SMS status</p></figcaption></figure>

**Table** ‎: SMS status

| **Status**   | **Description**                                             |
| ------------ | ----------------------------------------------------------- |
| Not Send     | Filter transactions have not sent SMS to buyers.            |
| Send Success | Filter successful sending-email transactions to the buyers. |
| Send Failed  | Filter failed sending-SMS transactions to the buyers.       |
| Don’t Send   | Do not send SMS or Email to the buyer.                      |



**3)  zDox Status:** Filter status upload files to zDox for the buyer, click ![](<../../.gitbook/assets/image (155).png>) (The detail describe below)

<figure><img src="../../.gitbook/assets/image (181).png" alt=""><figcaption><p>Filter - zDOX status</p></figcaption></figure>

**Table** ‎: SMS Status

| **Status**   | **Explanation**                                            |
| ------------ | ---------------------------------------------------------- |
| Not Send     | Items that have not yet been uploaded to the buyer's zDOX. |
| Send Success | Items successfully uploaded to the buyer's zDOX.           |
| Send Failed  | Item failed to upload to buyer's zDOX.                     |
