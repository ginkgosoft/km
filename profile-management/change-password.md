# Change Password

You can change the password by

1\) Click on the user login screen and select the menu “Change Password”.

<figure><img src="../.gitbook/assets/image (8).png" alt=""><figcaption><p>Change Password</p></figcaption></figure>

2\) The system will display a screen to fill in your current password, then click “Next”

<figure><img src="../.gitbook/assets/image (4).png" alt=""><figcaption><p>Screen asking for current password</p></figcaption></figure>

3\) Fill in a new password and confirm new password then click “Change Password”

<figure><img src="../.gitbook/assets/image (16).png" alt=""><figcaption><p>Screen input new password</p></figcaption></figure>
